Docutaz

A free and open source, cross-platform desktop GUI for MongoDB 8+, built on a modern mongosh shell.

dok·you·taz — doc (document) + dotaz, Czech for query

Download View on GitHub

Linux (x86_64 & arm64) · macOS (Apple Silicon) · Windows (x64)

What it does

Connect anywhere

Standalone, replica set or Atlas — with TLS/SSL and SSH-tunnel support.

Explore your data

Browse databases, collections and documents in a GUI tree, and page through large result sets.

Full mongosh shell

Run multi-statement JavaScript in a real mongosh tab with server-side autocomplete.

Edit visually

Insert, edit and delete documents with a visual editor — no hand-written JSON required.

Manage structure

Create and inspect indexes, manage users and collection settings.

Private by design

No telemetry or analytics. The only network traffic is your own MongoDB and SSH connections.

Getting started

Download the build for your platform from the latest release, then:

  • All platforms — install mongosh (2.x+); it isn't bundled. Keep it on your PATH or set its path in Options → Preferences.
  • Linux — install the distro Qt 6 / QScintilla runtime packages, extract, run ./docutaz.sh, then ./install-desktop.sh once for the app-menu launcher icon.
  • macOS (Apple Silicon) — unzip; the app is unsigned, so right-click → Open the first time.
  • Windows (x64) — extract and run docutaz.exe; on SmartScreen click More info → Run anyway.

Full step-by-step instructions are in the installation guide.
